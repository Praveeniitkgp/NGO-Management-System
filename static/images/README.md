# Logo Images Directory

Please place your logo images here:

- **logo.png** - Large logo for the hero section (home page). Recommended size: 384x384px or larger (will be displayed at 320x320px to 384x384px)
- **logo-small.png** - Small logo for the header/navigation. Recommended size: 40x40px (displayed at 40x40px)

Supported formats: PNG, JPG, JPEG, SVG

Once you add the images, they will automatically appear on the home page.
