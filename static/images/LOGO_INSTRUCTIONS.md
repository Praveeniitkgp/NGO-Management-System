# Logo Icon Instructions

## For Navbar Logo

Please place your logo image file here:
- **File name:** `logo-icon.png`
- **Location:** `/static/images/logo-icon.png`

### Image Specifications:
- **Format:** PNG (with transparent background recommended)
- **Size:** 40x40px (or larger, will be scaled down)
- **Aspect ratio:** 1:1 (square)
- **Background:** Transparent preferred

### The Logo Should Show:
- An open book with a heart shape integrated above its spine
- Blue color (#4285F4 or similar)
- Clean, minimalist design

### Alternative:
If you have the image in SVG format, you can name it `logo-icon.svg` and it will work the same way.

### Fallback:
If the image file doesn't exist, a fallback circular logo with "B" will be displayed automatically.
